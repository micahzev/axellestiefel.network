# axellestiefel.network

This repository contains the source code for _axellestiefel.network_

Visit the live site at:  
https://axellestiefel.network

Configured for [Firebase Hosting](https://firebase.google.com/products/hosting):

1.  Install the Firebase CLI (if needed):

    ```sh
    npm install -g firebase-tools
    ```

2.  Login to Firebase:

    ```sh
    firebase login
    ```

3.  Deploy to your Firebase project:

    ```sh
    firebase deploy
    ```

⚠️ Make sure the correct Firebase project is selected in your .firebaserc or use firebase use --add to link a project.
